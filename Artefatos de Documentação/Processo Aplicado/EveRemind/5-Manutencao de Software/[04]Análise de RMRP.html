Tipo da Modificação

Corretiva

Análise de Impacto

A mudança afetará somente o código fonte, pois só há necessidade de verificar as condições das novas senhas. Assim a mudança só estará se adequando ao RSEG2 não causando impacto aos outros componentes e partes do processo.

Impacto aos Usuários

Como a aplicação não foi liberada para usuários durante a descoberta da anomalia, não há impacto para os usuários.

Implicações/Limitações

Como implicação da mudança a aplicação fornecerá um maior nível de segurança para o usuário. Não há limitações.

Estimativa de Esforço

Para realizar a mudança será necessário: Recurso Humano: 1 integrante da equipe de desenvolvimento Tempo: 15 minutos de trabalho

Estimativa de Benefícios

A aplicação fornecerá um maior nível de segurança para o usuário.

Opções de Implementação

1- Aplicar uma verificação para analisar se a senha digitada está conforme os requisitos, informando ao usuário como deve ser uma nova senha.

Plano de Prevenção e Mitigação dos Riscos

Não há riscos por se tratar de uma mudança simples.
