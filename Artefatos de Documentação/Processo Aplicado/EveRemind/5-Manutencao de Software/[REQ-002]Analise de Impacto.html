[REQ-002] Requisição de Mudança/Relato de Problema.

Tipo da Modificação

Evolutiva.

Análise de Impacto

Esta requisição de mudança prevê uma alteração ao requisito já existente RFUN2.1.

Impacto ao Sistema e Usuários

Sempre que o endereço de e-mail primário e/ou secundário for modificado, o sistema enviará uma nova mensagem com uma ligação (link) de verificação para o (s) novo (s) endereço (s) eletrônico (s). Somente quando o (s) link (s) for (em) visitado (s), a (s) funcionalidade (s) associada (s) com a verificação deste (s) poderá (ão) ser utilizada (s).

Estimativa de Esforço para Implementação

Entidade Envolvida na Modificação Produto de Trabalho Afetado Horas
Equipe de Requisitos EOR 1
Equipe de Desenvolvimento Código Fonte 2

Estimativa de Benefícios

O Usuário deve ter liberdade para alterar todos os seus dados cadastrais em vez de apenas um subconjunto destes. Esta característica melhora a usabilidade do sistema para eventuais necessidades do cliente.

Opções de Implementação

1- Possibilitar a modificação das informações de email, implementando mecanismos para garantir a unicidade de novos endereços eletrônicos perante a base de dados do sistema.

Plano de Prevenção e Mitigação dos Riscos

Nenhum risco foi levantado durante a análise de impacto.
