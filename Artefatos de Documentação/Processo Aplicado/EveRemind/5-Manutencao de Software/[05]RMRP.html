Título

[05]RMRP - AIM002 - Relato de Anomalia

Descrição

Foi encontrada, durante as atividades de verificação, uma anomalia em relação à apresentação das informações na interface gráfica da aplicação.

Justificativa(s)/Consequência(s)

É necessária a mudança no código fonte para que corrija a apresentação de um nome muito extenso de uma atividade, pois esse deforma a interface da aplicação.
