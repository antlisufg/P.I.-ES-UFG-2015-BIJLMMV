[AIM-001] Requisição de Mudança/Relato de Problema.

Tipo da Modificação

Corretiva.

Análise de Impacto

A mudança afetará somente o código fonte, pois só há necessidade de verificar as condições das novas senhas. Assim a mudança só estará se adequando ao RSEG2 não causando impacto aos outros componentes e partes do processo.

Impacto ao Sistema e Usuários

Como a aplicação não foi liberada para usuários quando ocorreu a descoberta da anomalia, não há impacto para os usuários. O sistema por sua vez ficará mais adequado aos requisitos de segurança.

Estimativa de Esforço para Implementação

Entidade Envolvida na Modificação Produto de Trabalho Afetado Horas
Equipe de Desenvolvimento Código Fonte 0,5

Estimativa de Benefícios

A aplicação fornecerá um maior nível de segurança para o usuário e a correção deixará o sistema mais próximo de cumprir todos os requisitos não-funcionais estabelecidos. Especificamente no caso o RSEG2.

Opções de Implementação

1- Aplicar uma verificação para analisar se a senha digitada está conforme os requisitos, informando ao usuário como deve ser uma nova senha.

Plano de Prevenção e Mitigação dos Riscos

Não há riscos por se tratar de uma mudança simples.
